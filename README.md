# LaRT v2.0

LaRT v2.0 is a radiative transfer code for resonance-line scattering. In addition to Ly-alpha, it is designed to handle resonance and fluorescence scattering of a variety of metallic lines.

This version is currently being updated to support both RAMSES-based AMR outputs and more general AMR grid data. The AMR implementation is still under active validation, and its behavior should be checked more carefully before being used as fully verified production software.

## Status

- Support for AMR data from RAMSES is being developed and tested.
- Support for generic AMR grids is also being incorporated.
- The usage guide is still being prepared.
